Broccers

« L'app de gestion de la Brasserie Broc — personnel, carte, courses, IA. »

Outil interne pour la brasserie Broc (Puces du Canal, Villeurbanne). Tourne sur Mac Studio derrière Tailscale, accessible Chrome/Safari + iPhone PWA + tablette kiosk cuisine.

🔐 Tailscale only 🍻 Mac Studio 🖨️ PDF natif Dart 🧠 Claude Max cluster
4 features

Tout ce qu'il faut pour Broc, rien de plus.

Personnel, carte, courses, IA. Construit en moins d'un après-midi avec le framework SG de Stowy Agency.

👥

Personnel

Staff, planning, clock-in/out, pauses avec tracker droit du travail FR (alerte si pause < 20 min toutes les 6 h continues).

🍽️

Carte dynamique

Édition catégories + plats + allergènes (INCO). Versioning automatique. Export PDF A4 imprimable serveur-side (package Dart natif).

🛒

Liste de courses

Checklist par liste, par fournisseur. Urgences, quantités, unités. Toggle done en un tap depuis le téléphone en cuisine.

🧠

Pose question

Chat IA branché à Claude Max sur cluster Tailscale. Contexte : la carte courante + les courses ouvertes. Zéro cloud public.

L'architecture

Mac Studio → Browser. Rien d'autre.

Pipeline 100% maison, aucun cloud public. Le serveur tourne sur le Mac Studio, le browser parle via Tailscale, Claude tourne sur le cluster Stowy.

La stack

100% Dart, 100% chez toi.
